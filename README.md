# 🤖 Умный Telegram-бот с Mistral AI v2

Многофункциональный Telegram-бот, построенный на базе нейросети **Mistral AI**.  
Он понимает контекст беседы, ищет актуальную информацию в интернете, генерирует изображения, распознаёт голосовые сообщения, читает файлы, показывает погоду и делает краткие пересказы длинных текстов.

---

## Возможности

- **💬 Интеллектуальный диалог** – ответы с учётом истории разговора (Mistral Large).
- **🔍 Поиск в Google** – по ключевым словам *«найди», «расскажи о», «кто такой»*. Ответ формулируется как живая речь, без ссылок и маркдауна.
- **🎨 Генерация изображений** – фразы *«нарисуй», «сгенерируй», «покажи картинку»* запускают бесплатную нейросеть Pollinations.AI.
- **🎤 Распознавание голоса** – пришлите голосовое сообщение, бот расшифрует его с помощью локальной модели `faster-whisper` и ответит.
- **📄 Чтение и саммаризация файлов** – поддерживаются PDF, DOCX, TXT. Длинные документы автоматически сокращаются до 3‑5 предложений.
- **☀️ Прогноз погоды** – запросы *«погода в Москве», «сколько градусов в Питере»* обрабатываются через бесплатные API (Open‑Meteo + Nominatim).
- **🧠 Память диалога** – бот помнит контекст в рамках одной сессии (после перезапуска сбрасывается).
- **🎉 Приятные мелочи** – персональное приветствие, случайные комплименты и интересные факты.
